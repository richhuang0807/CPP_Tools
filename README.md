# CPP Tools — C++ 競賽模板產生器 & 簡易 IDE

專為競賽程式設計學習者打造的輕量級工具，快速生成常用模板並支援基本編輯與編譯功能。

---

## 專案介紹

CPP Tools 是一個以 Qt 6 開發的桌面應用程式，目標是讓競賽程式學習者能夠：

- 快速勾選所需的程式碼片段，一鍵產生乾淨的 C++ 模板
- 在內建編輯器中直接撰寫與修改程式碼
- 一鍵編譯並執行，不需要離開視窗切換到命令列

## 功能列表

### 模板產生器

勾選所需的片段，按下「生成」即可產生程式碼。

| 分組 | 元件 | 產生內容 |
|------|------|----------|
| 基礎配置 | `include_chx` | `#include<bits/stdc++.h>` |
| 基礎配置 | `std_chx` | `using namespace std;` |
| 基礎配置 | `fastio_chx` | `ios_base::sync_with_stdio(false)` + `cin.tie(NULL)` |
| 型別與宏 | `define_chx` | `INF`, `MAXN`, `MOD` 常數定義 |
| 型別與宏 | `ll_chx` | `#define ll long long` |
| 型別與宏 | `for_chx` | `#define rep` + `#define per` 迴圈宏 |
| 常用函式 | `gcd_chx` | 遞迴 GCD 函式 |
| 常用函式 | `qpow_chx` | 快速冪函式 |
| 常用函式 | `fib_chx` | 費氏數列函式 |
| 結構模板 | `main_chx` | `int main()` 開頭與結尾 |
| 結構模板 | `testcase_chx` | 多筆測資輸入（T 筆）|
| 結構模板 | `eof_chx` | EOF 輸入模式 |

> 注意：`testcase_chx` 與 `eof_chx` 互斥，同時勾選會跳出提示詢問。

---

### 操作按鈕

| 按鈕 | 功能 |
|------|------|
| 生成 | 根據勾選狀態產生程式碼到編輯器 |
| 清除 | 清空所有勾選與編輯器內容 |
| 複製 | 全選並複製編輯器內容到剪貼簿 |
| 下載 | 另存新檔（每次都跳出路徑選擇視窗） |

---

### IDE 功能

| 按鈕 | 功能 |
|------|------|
| 開啟 | 開啟現有 `.cpp` 檔案到編輯器 |
| 存檔 | 第一次存時選路徑，之後直接存到同一路徑 |
| 上一步 | 復原上一步編輯 |
| 下一步 | 取消上一步復原 |
| 編譯 | 呼叫 `g++` 編譯，若有錯誤會顯示訊息 |
| 執行 | 在新的 cmd 視窗執行已編譯的程式 |
| 編譯並執行 | 編譯成功後自動在 cmd 視窗執行 |
| 主題切換 | 切換深色 / 淺色主題 預設淺色|

## 安裝與執行

### 需求

- Windows 作業系統
- 已安裝 MinGW g++ 並加入系統 PATH
```
cpptools/
├── mainwindow.h        # 主視窗標頭檔（成員變數、Slot 宣告）
├── mainwindow.cpp      # 主視窗邏輯（所有 Slot 實作）
├── mainwindow.ui       # Qt Designer UI 檔案
├── main.cpp            # 程式進入點
└── README.md           # 說明文件
```

## 使用流程

### 快速生成模板

1. 在左側勾選所需的程式碼片段
2. 按下「生成」按鈕
3. 程式碼出現在右側編輯器
4. 按「複製」複製到剪貼簿，或「下載」存成檔案

### 編譯並執行

1. 在編輯器撰寫或生成程式碼
2. 按「存檔」選擇儲存路徑
3. 按「編譯並執行」
4. 若有錯誤會跳出提示視窗顯示錯誤訊息
5. 若編譯成功會自動開啟 cmd 視窗執行程式

## 主題

- 預設為**淺色主題**
- 按右上角「主題」按鈕可切換深色/淺色

## 已知限制

- 目前僅支援 Windows 平台
- 編譯需要系統已安裝 g++ 並設定 PATH
- 不支援語法高亮（未來可考慮使用 `Qscintilla`）
- 不支援 Tab 自動補全
